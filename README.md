# Социальная сеть САФУ

Веб-приложение социальной сети для студентов и преподавателей Северного (Арктического) федерального университета имени М.В. Ломоносова.

## 🚀 Возможности

- 📝 Создание постов с изображениями
- 💬 Многоуровневые комментарии (до 3 уровней вложенности)
- 📨 Личные сообщения между пользователями
- 👥 Система друзей с заявками
- 🔔 Система уведомлений
- 🎨 Светлая и тёмная темы оформления
- 🔍 Поиск пользователей
- 🛡️ Автоматическая цензура нецензурной лексики
- 📱 Адаптивный интерфейс для мобильных устройств

## 🛠 Технологии

- **Backend**: Python 3.12, Django 4.2.7
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **База данных**: SQLite (с возможностью миграции на PostgreSQL)
- **Дополнительно**: Django Allauth, Pillow, Django Channels

## 📦 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Halkerov/social-mvp-2026.git
cd social-mvp-2026
```

2. Создайте виртуальное окружение:
```bash
python3 -m venv venv
source venv/bin/activate  # для macOS/Linux
# или
venv\Scripts\activate  # для Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Примените миграции:
```bash
python manage.py migrate
```

5. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

6. Запустите сервер:
```bash
python manage.py runserver
```

7. Откройте в браузере: http://127.0.0.1:8000/
